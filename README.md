# 🛒 Conversion Prediction for E-commerce
## 🎯 Задача
Предсказать, совершит ли пользователь покупку в рамках сессии интернет-магазина
(`Revenue = 1`) или нет (`Revenue = 0`) на основе его поведения.
---
## 💡 Зачем это бизнесу
Решение задачи позволяет:
* 🎯 таргетировать пользователей с высокой вероятностью покупки
* 📈 увеличить конверсию за счёт персонализированных предложений
* 💰 снизить маркетинговые расходы (не тратить бюджет на “холодных” пользователей)
---
## 📊 Данные
Датасет содержит **сессионные данные пользователей**:
* 🧭 поведение на сайте (просмотры страниц, время)
* 📉 метрики вовлечённости (BounceRate, ExitRate)
* 🛍 взаимодействие с товарами
* 🌍 контекст пользователя (регион, устройство, трафик)
**Целевая переменная:**
* `Revenue` — факт покупки
---
## 🔍 Что сделано
### 1. Data Quality & EDA
* проверка пропусков и дубликатов
* анализ конверсии
* поиск факторов, влияющих на покупку
---
### 2. Feature Engineering
* масштабирование числовых признаков
* one-hot encoding категориальных
* подготовка данных через `ColumnTransformer`
---
### 3. Моделирование
Сравнение моделей:
* Dummy (baseline)
* Random Forest
* XGBoost
---
### 4. Тюнинг
* подбор гиперпараметров для XGBoost
---
### 5. Оценка
Использованы метрики:
* ROC-AUC
* PR-AUC
* Accuracy
---
## 🧠 Основные инсайты
* 📌 Поведение пользователя внутри сессии (особенно страницы товаров) — ключевой фактор
* 📉 Высокий bounce rate снижает вероятность покупки
* 🔁 Returning users конвертируются лучше
---
## 🚀 Как использовать результат
Модель можно применять для:
* 🎯 таргетинга пользователей с высокой вероятностью покупки
* 💸 оптимизации рекламных кампаний
* 📬 персонализации офферов
